BAR PLOT

It helps us to show the relation between a categorical variable (x-axis) and a Numerical varialbe (y - axis )

In[2]:
import seaborn as sns 
import matplotlib.pyplot as plt 

tips = sns.load_dataset("tips")

plt.figure(figsize=(8,6))
sns.barplot(x="day", y="total_bill", data=tips, palette="Set2")

plt.title("Average Total Bill by Day", fontsize=16)

plt.show()

2. COUNT PLOT

--It is used to show counts of observations in each category.

--It is Similar to a histogram but for a categorical data.

In[4]:
plt.figure(figsize=(8,6))

sns.countplot(x="day", data=tips, palette="Set3")
plt.title("Number of Observations by Day", fontsize=16)
