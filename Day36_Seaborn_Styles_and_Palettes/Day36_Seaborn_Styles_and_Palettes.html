1. Seaborn Styles

Seaborn provides built in themes to control the overall look of plots

For example

"darkgrid" this is default

"whitegrid"

"dark"

"white"

"ticks"

Syntax => sns.set_style('whitegrid')

2.Color Palettes

With the help of seaborn we can easily set custom colors:

"deep", "muted", "pastel", "bright", "dark", "colorblind"

and custom palette() also

3.Combining Styles and Colors

Let us see how the same data looks completely different with just a change in theme and palette.

Now we will practice of the above functions

In[2]:
import seaborn as sns
import matplotlib.pyplot as plt 

tips = sns.load_dataset ("tips")

for style in ["darkgrid", "whitegrid", "dark", "white", "ticks"]:
    sns.set_style(style)
    sns.scatterplot(x="total_bill", y="tip", data=tips)
    plt.title(f"Style: {style}")
    plt.show()


palettes = ["deep", "muted", "pastel", "bright", "dark", "colorblind"]
for pal in palettes:
    sns.set_palette(pal)
    sns.histplot(tips["total_bill"], bins=20, kde=True)
    plt.title(f"Palette: {pal}")
    plt.show()


sns.set_theme(style="whitegrid")
sns.scatterplot(
    x="total_bill", y="tip", 
    hue="size", size="size", 
    palette="coolwarm", sizes=(20, 200),
    data=tips)

plt.title("Custom Coolwarm Palette with Size Encoding")
plt.show()
